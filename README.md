## 1. What is BMI?
BMI = Weight (kg) / (Height in meters)²

It tells you whether you're:

* Underweight

* Normal

* Overweight

* Obese

# 2. Example Output:

Enter your weight in kg: 65  
Enter your height in meters: 1.75  

Your BMI is: 21.2  
You are in the Normal weight category.

# 3. Categories:
## BMI Range -  Category
* < 18.5	   -  Underweight
* 18.5–24.9	 -  Normal weight
* 25–29.9	   -  Overweight
* 30+	       -  Obese
